# Fresnel 计算器使用说明

计算多层膜在给定波长与入射角下的 **反射率 R**、**透射率 T** 以及 Fresnel 系数（r, t），并支持 TE/TM 偏振。

## 基本流程

1. **设置参考波长与入射角**：在「膜系结构」页配置参考波长（μm）和入射角度（度）。
2. **构建膜系**：在「多层膜构建指令」中输入或编辑膜层序列（见下文语法）。
3. **刷新表格**：点击「🔄 刷新」将指令解析为膜层表格，可按需在表格中微调材料、厚度或覆盖 n/k。
4. **计算**：点击「▶️ 计算」得到 R、T 及膜系结构图。

---

## 多层膜构建指令

以下为「多层膜构建指令」的书写规则，用于快速定义从入射侧到基底侧的膜层序列。

### 基本单元：材料 + 厚度

- 每层由 **材料名** 和 **厚度（μm）** 组成，用空格分隔。
- 示例：`SiO2 0.1` 表示 0.1 μm 的 SiO2；`Vacuum 0` 表示 0 厚度的真空（常用于入射与出射介质）。

### 可选：覆盖折射率 n、k

- 若需覆盖材料库的 n/k，可在该层厚度后追加 **n** 和 **k** 两个数（空格分隔）。
- 格式：`材料 厚度 n k`
- 示例：`Ta2O5 0.05 2.1 0.001` 表示 0.05 μm 的 Ta2O5，并使用 n=2.1、k=0.001。

### 括号周期重复

- 用括号 `( ... )` 包裹若干层，再写 `^数字` 表示该序列重复多少次。
- 示例：`(SiO2 0.1 Ta2O5 0.01)^5` 表示 (SiO2 0.1 μm, Ta2O5 0.01 μm) 重复 5 次，共 10 层。

### 完整示例

- 单层：`Vacuum 0 SiO2 0.1 Vacuum 0`
- 多层周期：`Vacuum 0 (SiO2 0.1 Ta2O5 0.01)^5 Vacuum 0`
- 带 n/k 覆盖：`Vacuum 0 (SiO2 0.1 1.45 0 Ta2O5 0.01 2.1 0.001)^3 Vacuum 0`
- 如果材料在 Material Database 中被添加, 可以不显示指定材料 nk  
假定 SiO2/Ta2O5已经添加到了数据库中： `Vacuum 0 (SiO2 0.1 Ta2O5 0.01 )^3 Vacuum 0` 。
